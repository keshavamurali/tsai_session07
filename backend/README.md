# Backend for the Chrome extension. 

## It contains following files
1. main.py: Agent as well as the entry point
2. mcp_server.py : Code that contains document processing and embedding generation. Also has the search_documents which cna be exposed as MCP tool.
3. perception.py: Perception logic
4. memory.py : Memory retrieval
5. decision.py: Decision maker
6. action.py: Tool calling
